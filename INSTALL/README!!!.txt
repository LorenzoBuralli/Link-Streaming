ENG

Before starting the application, if you haven't already installed Python on your operating system, use "Py 3.11.3 + Lib Auto Install.bat" to start an automatic installation of the interpreter and the two libraries needed to run the program.
If you wish to uninstall it, use the other "Unistall.bat" script to initiate a clean uninstall of Python 3.11.3 and the two libraries.
If you wish, you can verify the installed versions with "Verify.bat".

Known Issues:
-The program doesn't work properly without Python version 3.11.3;
-The program doesn't work correctly without the version of the libraries listed in "requirements.txt";
-The program does not work properly without the version of "chromedriver.exe" compatible with that of the Chrome browser. Downloadable from "https://chromedriver.chromium.org/downloads" if it needs to be updated.
